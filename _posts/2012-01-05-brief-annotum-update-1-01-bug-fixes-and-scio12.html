Firstly, allow me to wish a very Happy New Year to everyone reading this.  The response to the launch of Annotum v1.0 has been very enthusiastic, and a number of you have launched journals and other sites based on your Knol content.  Some others are in the process of setting up new journals -- all of which is very exciting.

Just before the holidays, we made a fair bit of progress in squashing some of the initial bugs that were found in Annotum v1.0, and those changes have all been deployed to a new branch on GitHub.  For the next couple of weeks we'll be continuing to implement the last fixes for what will become v1.01.  Once fully tested, we'll update the downloads on GitHub and update the main theme code on WordPress.org. Of course, if you can't wait, you can always grab the latest branch code as a zip download.

We hope to release v1.01 prior to ScienceOnline 2012 later this month, where I'll be presenting an Annotum Demo.

In the meantime, please feel free to use the support page to seek help, refer to the knowledge base for tutorials and answers, and contact us if you need help setting up a customized Annotum site or have any other questions not covered by the above.
